## Funcionalidades
- Usuário faz login
- Usuário pertence a uma empresa
- Empresa pode criar checklists
- Checklist tem itens
- Usuário executa checklist por dia
- Sistema salva histórico

## Fora do escopo
- Pagamentos
- Notificações
- Relatórios avançados

## Stack
- Backend: NestJS
- Frontend: React
- Banco: PostgreSQL
